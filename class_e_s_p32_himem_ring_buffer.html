Detailed Description

template<typename T>
class ESP32HimemRingBuffer< T >

Ring buffer implementation using ESP32's extended high memory (himem) API.

ESP32HimemRingBuffer extends the ESP32HimemBuffer to provide circular buffer functionality, where read and write operations wrap around when reaching the buffer's end. This is ideal for:

  • Continuous audio streaming applications
  • Producer-consumer scenarios with different read/write rates
  • Real-time audio processing with minimal latency
  • Efficient memory usage for continuous data flows
