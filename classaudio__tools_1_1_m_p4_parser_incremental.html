Detailed Description

MP4ParserIncremental is a class that extends MP4Parser to support incremental parsing of MP4 boxes.

It allows for processing boxes as they are received, which is useful for large files or streaming scenarios. It provides a callback mechanism to process box data incrementally. The default callback prints the box information to Serial.
